## What is this?
EngineGL-2D is a 2D sprite engine written in C++ and based on __OpenGL__ with __GLFW__, using __stb_image__ for reading image data.
The engine uses image data arranged in sprite sheets, which are associated in groups and used through objects called Batches.
It also supports UI based on arrangements of widgets using primitive positioning and layout rules and is capable of I/O operations.

As of now, there is no support for audio playback.
Sample code will be included in the future in a special folder with examples.

This project requires the following to begin the build process:
- CMake(minimum version 3.20).
- vcpkg
